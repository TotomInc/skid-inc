v0.08: kill these bugs! feedback update! (TBD)

  • Changed/added some commands for their respective content.
  • Server upgrades now display the proper price.
  • Some QoL changes (team -status display only hired hackers).

v0.07: the first version released to alpha-testers! (5 july 2016)

  • Better guide.
  • Start-game is now easier (the click reducer is now at 5), and automation is faster to get (hackers are less expensive).
  • Hackers now have a required level to be hired.
  • Fixed some bugs.

v0.06: the first big balancing update! (26 june 2016)

  • Fixed bugs with team/hackers command.
  • Balancing changes: places give less exp and take slightly more time, quickhack server cost reduced.
  • You can't simple hack if you are hacking a place, and added a command to cancel the current place hack.
  • Changed notification icon.
  • Fixed some typos.

v0.05: not important update. (10 june 2016)

  • Added server upgrades.
  • Redone code for commands.
  • Reduced auto-save time (from 1 sec to 30 sec).
  • Compressed save to local-storage (thanks to Pullingour), using LZString.
  • Reduced cooldown time between two commands.
  • Better tutorial.

v0.04: matrix update! (7 june 2016)

  • Added hackers to automatically hack places for you.
  • Added abilities to enhance your hacking power.
  • Added a notifiaction system (will alert you when you are not in browser).
  • Added Matrix background effect (this effect is really beautiful, take a look at it!).
  • More balancing to the game.

v0.03: NSA update (save system)! (3 june 2016)

  • Added save system to the game (seems stable and will persists through all versions).
  • Big balancing update: game is now a lot harder.
  • Servers code totally redone (it was a spaghetti code!).
  • Lot of bugs are fixed.

v0.02: the sounds update. (2 june 2016)

  • Added sounds and ambient background music/sound.
  • Added achievements with achievements points.
  • Added more places to hack, new servers types.
  • Better UI.
  • Some backend changes.
  • Fixed lot of bugs.
